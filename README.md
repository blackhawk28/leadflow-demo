LeadFlow AI — Real Estate Lead Follow-Up System
A live demo of an AI automation system that responds to new real estate leads in under 60 seconds — automatically generating personalized follow-up emails, agent notifications, and call scripts using Claude AI.
Live Demo
👉 Try it here
What it does

Lead fills an intake form (simulating Zillow, Realtor.com, website forms)
Claude AI reads the lead details instantly
Generates a personalized follow-up email referencing their budget, area and timeline
Creates an internal agent notification with urgency score
Produces a ready-to-use 60-second call script

Tech Stack
Claude AI API · JavaScript · HTML/CSS · REST API
